### Issue 2: Claude Code Extension Availability & Login

**Problem:**
- The exact extension named *“Claude Code”* was not clearly available in the Cursor marketplace
- Multiple similar extensions were listed, creating confusion
- After installing the closest official match (*Claude Code for VS Code by Anthropic*), there was no clear login interface
- No authentication popup appeared, and command palette did not trigger login

**Approach:**
- Evaluated available extensions and selected the closest official version
- Attempted login using:
  - Command palette
  - Extension interface
  - Restarting the IDE

**Conclusion:**
The exact extension mentioned in the instructions was not directly identifiable, and the authentication flow was not clearly accessible. Instead of getting blocked, I proceeded with the closest available solution and documented the limitation.
